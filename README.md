# kobin-linebot

Line Echo Bot in Kobin.

![DEMO](demo.gif)


## Setup

```console
$ python3.6 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ wsgicli run app.py app -p 8080
```

## ngrok

ngrok for https.

refs: https://ngrok.com/doc

```console
$ ngrok http 8080
```
